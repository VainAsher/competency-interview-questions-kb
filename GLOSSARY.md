# Glossary

Shared terms for the knowledge base. Only the orchestrator edits this file;
workers return new terms in their JSON reply. `Source key` refers to
`SOURCE_REGISTRY.md` (`-` = general usage).

| Term | Definition | Source key |
|------|------------|------------|
| Competency-based interview | A structured interview in which candidates evidence defined competencies/behaviours with examples from past experience, scored against set criteria. | - |
| STAR | Situation, Task, Action, Result — the standard structure for evidencing a competency example. | - |
